
# Simple Scraping

Simple explanation to understand how to make a simple scraping
## Run Locally

Clone the project

```bash
  git clone https://github.com/SexyPy/simple_scraping.git
```

Go to the project directory

```bash
  cd simple_scraping
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the scraping

```bash
  py __main__.py
```
